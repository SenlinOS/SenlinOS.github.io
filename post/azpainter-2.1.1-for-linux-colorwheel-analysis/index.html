AzPainter 2.1.1 for Linux 的色环面板解析

AzPainter 2.1.1 for Linux 加入了 HLS 颜色滑片，它与色环面板可以密切配合！ HLS 滑片的颜色走势符合绘画的色彩原理，可以使用它来观察色环面板。

AzPainter 色环上的冷色与暖色：


如图所示的颜色位置，色环分成两部分。当然具体颜色的冷暖也得看是什么颜色放在一起做对比。 可以看到分界线是从三角形的黑色端点到对边的垂直线，非常容易找到。

AzPainter 色环上的亮色与暗色：


同样是色环颜色被分成亮暗两部分，可以凭感觉体会下。

AzPainter 2.1.1 for Linux 的色环面板中的明度与饱和度的拾色有两个选择：
HSV 三角形 与 HSV 正方形，我更喜欢 HSV 三角形，因为三角形中的颜色走势更有规律。

可以拖动 HLS 颜色滑片来观察 HSV 三角形 与 HSV 正方形 中的颜色规律。
HLS 颜色滑片：H = 色相、L = 明度 、S = 饱和度

明度的走势：


从暗到亮，饱和度不变。 三角形中颜色由暗点以直线行进，到三角形水平中线后又以直线行进到亮点，走势非常有规律。 正方形中的明度走势也可以说有些规律，但没三角形中的明度走势好掌握。

饱和度的走势：


图中箭头方向，饱和度逐渐变大。明度保持不变。 三角形中的饱和度走势是水平移动，非常有规律～便于操作。 正方形中的饱和度走势可以说很难掌握规律！

这就是我喜欢使用 HSV 三角形 选择颜色的原因！

在这里还是要感谢 AzPainter for Linux 的作者 Azel 实现了这个好用的色环颜色面板！
